# Healthcare-Fraud-Detection-EDA-->ML
Healthcare fraud detection project using machine learning on insurance claims data. Includes EDA, data preprocessing, feature engineering, and classification models to identify fraudulent claims. Handles imbalanced data and analyzes key fraud indicators like claim amount, provider behavior, and submission patterns.

# 🏥 Healthcare Fraud Detection

## 📌 Project Overview
This project focuses on detecting fraudulent healthcare insurance claims using machine learning techniques. Fraud detection is critical for reducing financial losses and improving system efficiency.

---

## 📊 Dataset
- 10,000 healthcare insurance claims
- Includes patient, provider, financial, and temporal features
- Target variable: `Is_Fraud` (0 = Legitimate, 1 = Fraud)
- Imbalanced dataset (~8% fraud cases)

---

## ⚙️ Features Used
- Claim Amount & Approved Amount
- Insurance Type
- Provider Specialty
- Patient Demographics
- Claim Submission Details
- Visit Type & Length of Stay

---

## 🔍 Exploratory Data Analysis (EDA)
- Fraud vs Non-Fraud distribution
- Claim amount patterns
- Provider-wise analysis
- Missing value analysis

---

## 🛠️ Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature transformation

---

## 🤖 Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

---

## 📈 Evaluation Metrics
- Precision
- Recall
- F1-Score
- Confusion Matrix

---

## 🔑 Key Insights
- High claim amounts are more likely to be fraudulent
- Faster claim submission shows suspicious behavior
- Certain providers have higher fraud rates

---

## 🚀 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

---

## 📌 Conclusion
The model helps identify potentially fraudulent claims and can assist insurance companies in improving fraud detection systems and reducing financial losses.
